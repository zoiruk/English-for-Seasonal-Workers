# 💬 Разговорник на каждый день

Готовые фразы, которые работают сразу. Учите целыми фразами — не разбирайте на слова.

## Приветствие и прощание

| English | Перевод |
|---------|---------|
| Hello! / Hi! | Здравствуйте! / Привет! |
| Good morning / afternoon / evening | Доброе утро / день / вечер |
| How are you? | Как дела? |
| I'm fine, thanks. And you? | Хорошо, спасибо. А ты? |
| Goodbye! / Bye! / See you! | До свидания! / Пока! / Увидимся! |
| See you later / tomorrow. | Увидимся позже / завтра. |
| Have a nice day! | Хорошего дня! |

## Знакомство

| English | Перевод |
|---------|---------|
| What's your name? | Как тебя зовут? |
| My name is… / I'm… | Меня зовут… / Я… |
| Nice to meet you. | Приятно познакомиться. |
| Where are you from? | Откуда ты? |
| I'm from Russia. | Я из России. |

## Вежливость

| English | Перевод |
|---------|---------|
| Please. | Пожалуйста (просьба). |
| Thank you. / Thanks a lot. | Спасибо. / Большое спасибо. |
| You're welcome. | Пожалуйста (ответ). |
| Sorry. | Извините (виноват). |
| Excuse me. | Простите (обратиться). |
| No problem. | Без проблем. |

## Когда не понял (САМОЕ ВАЖНОЕ для начинающих!)

| English | Перевод |
|---------|---------|
| Sorry, I don't understand. | Извините, я не понимаю. |
| Can you repeat that, please? | Повторите, пожалуйста. |
| Can you speak slowly, please? | Говорите медленнее, пожалуйста. |
| What does … mean? | Что значит …? |
| How do you say … in English? | Как сказать … по-английски? |
| I don't know. | Я не знаю. |
| Just a little. | Совсем чуть-чуть. |

## В магазине / кафе

| English | Перевод |
|---------|---------|
| How much is it? | Сколько стоит? |
| Can I have …, please? | Можно мне …? |
| I'd like … | Я бы хотел … |
| Here you are. | Вот, пожалуйста. |
| Can I pay by card? | Можно оплатить картой? |

## В путешествии

| English | Перевод |
|---------|---------|
| Where is the toilet / station? | Где туалет / вокзал? |
| How do I get to …? | Как пройти к …? |
| Can you help me? | Можете мне помочь? |
| I'm lost. | Я заблудился. |
| Is there Wi-Fi here? | Здесь есть Wi-Fi? |

## Полезные мелочи

| English | Перевод |
|---------|---------|
| Yes / No / Maybe | Да / Нет / Может быть |
| OK / Sure | Хорошо / Конечно |
| Of course. | Конечно. |
| I think so. | Думаю, да. |
| Me too. | Я тоже. |
| Don't worry. | Не волнуйся. |
