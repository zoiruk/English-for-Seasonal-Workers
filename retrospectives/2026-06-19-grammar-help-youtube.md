# Retro: «Не понял? Объясню ещё проще» + YouTube-резерв — 2026-06-19

## Что построено
Движок помощи «ещё проще» в конце grammar-таба + эталонный L1. Коммит `01e9ed1`.

- **Движок** (`app/app.js`): `helpBlock(g)` рендерит свёрнутую лесенку эскалации после
  карточки примеров — тоггл (closed по умолчанию, `aria-expanded`) → формула + 1–2 мини-примера
  (TTS переиспользует общий `[data-spk]`-враинг) → мелкая последняя-резервная YouTube-ссылка
  (`results?search_query=` + `encodeURIComponent(ytQuery)`, `target=_blank rel=noopener`,
  «нужен интернет») → инструкция возврата. Тоггл-хендлер — в `wireBody` (ветка grammar).
- **UI-строки** (`UI.ru`): `help_toggle/help_formula/help_yt/help_yt_offline_hint/help_return`,
  формальный вы-тон как в остальном UI.
- **Аудит** (`scripts/audit.js`): опц. `grammar.simple_ru` (formula непустая, examples ≥1 с
  en/transcr/ru) + `ytQuery` (строка). **Снежок** (`scripts/snowball.js`): проверяет
  `simple_ru.examples[].en`. Оба поля опциональны → урок без них валиден.
- **Контент L1**: формула «Кто + am/is/are + слово» + 2 снежок-чистых примера
  (`I am a worker` / `He is a farmer`, разбор по словам в `ru`) + RU `ytQuery`.
- **Стиль** (`app/style.css`): вторичная YouTube-ссылка, touch ≥44px, light-only.
- Бамп `CACHE_VERSION` `esw-v17`→`esw-v18`.

## Решения (и почему)
- **Внутри-урочная помощь — основной путь, YouTube — последний резерв за тогглом.** Сохраняет
  offline-first и контроль контента; видео неконтролируемо, поэтому только аварийный выход.
- **`ytQuery` = ручной RU-запрос, не выверенный ролик.** Единственный рычаг на неконтролируемом
  механизме — формулировка запроса. Если всплывёт плохая выдача — меняется `ytQuery` без правки движка.
- **Транскрипции примеров переиспользованы дословно** из уже сверенных вхождений L1
  (`worker`→уёкэ, `farmer`→фармэ) → `verify-transcr`/`verify-ipa` не затронуты (новых слов нет).
- **Формула опускает артикль `a`** («+ слово», хотя примеры `a worker`) — осознанное упрощение
  ради «проще»; зафиксировано как косметическая заметка ревью, не баг.

## Процесс
3-слойная проверка пройдена: (1) `node --check` + `npm run audit` зелёный; (2) ручной чек-лист
(фермерская лексика, RU-only, реальные `ru`-разборы); (3) адверсариальный ревьюер — 5/5 PASS.
Превью проверено через `preview_eval` (скриншоты нельзя — таймаут на `backdrop-filter`):
тоггл свёрнут/раскрывается/сворачивается, aria корректна, YT-ссылка `target=_blank` с
encoded RU-запросом, консоль чистая.
