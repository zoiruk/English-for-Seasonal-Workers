# 🇬🇧 English from Zero — полный курс английского языка

Структурированный курс английского языка **с нуля (A0)** до уверенного **A2/B1**.
Для русскоязычных. Каждый урок: объяснение на русском → английский материал
с транскрипцией и переводом → упражнения → ответы.

> Цель курса: одновременно развивать **разговорную речь, грамматику, словарный
> запас, чтение и аудирование**, плюс базовый **английский для работы и IT**.

---

## 📚 Как устроен курс

```
.
├── README.md                  ← вы здесь (карта курса + инструкция)
├── PROGRESS.md                ← чек-лист вашего прогресса
├── HOW-TO-STUDY.md            ← как заниматься, чтобы был результат
├── lessons/A1/                ← основные уроки (грамматика + лексика)
├── vocabulary/                ← списки слов и разговорник
├── tracks/                    ← доп. треки: разговор и Work/IT English
└── app/                       ← интерактивное веб-приложение (карточки + квизы)
```

### Два способа заниматься

1. **Читать уроки** в папке [`lessons/`](lessons/A1/) — прямо здесь, в GitHub или
   в любом редакторе. Каждый урок самодостаточен.
2. **Тренироваться в приложении** — откройте [`app/index.html`](app/index.html)
   в браузере. Там карточки (flashcards), квизы и сохранение прогресса.

## 🚀 С чего начать

1. Прочитайте [`HOW-TO-STUDY.md`](HOW-TO-STUDY.md) — 5 минут, сэкономит месяцы.
2. Откройте [Урок 00 — Введение](lessons/A1/00-introduction.md).
3. Открывайте [приложение](app/index.html) после каждого урока для тренировки.
4. Отмечайте прогресс в [`PROGRESS.md`](PROGRESS.md).

### Как открыть приложение

Просто откройте файл `app/index.html` двойным кликом в браузере.
Либо запустите локальный сервер:

```bash
cd app && python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## ⏱️ Сколько это займёт

При занятиях **20–30 минут в день**:
- 1 урок = 2–3 дня (теория + упражнения + карточки).
- Весь уровень A1 ≈ **6–8 недель**.

Главное — регулярность, а не объём. Лучше 15 минут каждый день, чем 3 часа раз в неделю.
