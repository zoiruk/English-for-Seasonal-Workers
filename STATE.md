**Построено и проверено (аудит зелёный):**

  квиз-тренажёр + streak + PWA.
- **Уроки 1–9 «золотые»:** у всех есть `everyday` (функц. блок); в `quiz` типы `[LISTEN]`
  (аудирование: ▶ TTS → выбрать перевод, англ. текст скрыт) и `[GIST]` (чтение: RU-вопрос по диалогу).
  L1 — am/is/are + национальности (35 слов); L2 — притяжательные my/your/his/her; L3 — this/these + a/an;
  **L4 — числа/время/дни/деньги £; L5 — места + предлоги (Where is…?); L6 — Present Simple + do/does +
  наречия частоты + объектные me/him/us/them** (закрыта отложенная дыра); **L7 — Present Continuous +
  будущее-план (am/is/are + -ing; «I am starting tomorrow»); 30 новых -ing-глаголов (spray/sweep/read/
  climb/feed…), будущее и учится, и проверяется в quiz**; **L8 — have/has + do/does + одежда/PPE + цвета
  (30 слов); плейн `have/has` БЕЗ `have got`/сокращений (решение грилла — не трогать токенайзер; `have got`
  узнаваемой пометкой); quiz-фокус на согласовании he→has, does+have-база**; **L9 — there is/there are +
  some/any + комната/мебель/техника вагончика (room/bed/shower/heater/kettle/fridge…) + 8 состояний-прилагательных
  (broken/wet/cold/hot/heavy/empty/full/ready через is/are-предикатив); плейн `is not`/`are not`; закрыта дыра
  state-прилагательных, перенесённая из L8.** **L10 — how much/how many + would like (вежливое «хочу») +
  еда/напитки/покупки Tesco (bread/milk/rice/eggs/tin/biscuit/pasta/honey… 30 слов BrE); все ценовые вопросы —
  на неисчисляемых/ед.ч. (much↔many без скрытого price-исключения, чтобы не конфликтовать с ключом quiz);
  плейн `would not like` + «No, thank you»; диалог «магазин» worker↔cashier с сходящимся итогом-корзиной.**
  Снежок-whitelist по урокам — в `scripts/lib.js` (`WHITELIST[1..10]`, L10 добавил `many/would/like/of`).
