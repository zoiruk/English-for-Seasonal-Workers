# Урок 03 — Предметы вокруг: артикли и множественное число

> Грамматика: артикли **a / an / the**, множественное число, **this/that/these/those**.
> Тема: предметы вокруг нас.

---

## 1. Артикли a / an (один, какой-то)

В английском перед предметом в единственном числе обычно стоит «маленькое слово» — артикль.

**a / an** = «один, какой-то» (предмет неопределённый, упоминается впервые):
- **a** — перед согласным звуком: a cat, a book, a dog.
- **an** — перед гласным звуком: an apple, an egg, an hour.

> Выбор a/an зависит от **звука**, а не буквы: an **h**our [ан ауэ] — h не читается.

Примеры:
- **It's a phone.** — Это телефон.
- **I have an apple.** — У меня есть яблоко.

---

## 2. Артикль the (этот, конкретный)

**the** [зэ] = «тот самый, конкретный» (о котором уже знаем):

- I have **a** dog. **The** dog is big. — У меня есть собака. (Эта) собака большая.
- Open **the** door, please. — Открой (эту) дверь, пожалуйста.

> 🔑 Логика: первый раз — **a/an** (какой-то), второй раз — **the** (тот самый).

---

## 3. Множественное число: + s

Чтобы сделать «много», обычно добавляем **-s**:

| Один | Много |
|------|-------|
| a book | books — книги |
| a cat | cats — кошки |
| a friend | friends — друзья |

**Особые случаи:**
- слова на -s, -ss, -sh, -ch, -x → **+es**: bus → bus**es**, box → box**es**.
- слова на согласную + y → **y меняется на ies**: city → cit**ies**, baby → bab**ies**.
- неправильные (запомнить!):

| Один | Много |
|------|-------|
| man (мужчина) | **men** |
| woman (женщина) | **women** [уимин] |
| child (ребёнок) | **children** |
| person (человек) | **people** [пипл] |
| foot (нога-ступня) | **feet** |
| tooth (зуб) | **teeth** |

---

## 4. This / That / These / Those (этот / тот)

| | Один | Много |
|--|------|-------|
| **близко** (этот/эти) | **this** [зис] | **these** [зиз] |
| **далеко** (тот/те) | **that** [зэт] | **those** [зоуз] |

- **This is my book.** — Это моя книга. (рядом)
- **That is your car.** — Та машина — твоя. (вдалеке)
- **These are my friends.** — Это мои друзья. (рядом, много)
- **Those are big houses.** — Те дома большие. (вдалеке, много)
