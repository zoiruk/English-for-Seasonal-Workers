# 5 эталонных источников — якорь натуральности и охвата

> **Зачем файл существует.** Риск №1 проекта: автор контента — ИИ, приёмщик — владелец с
> нулевым английским, а все авто-проверки (audit/snowball/verify-ipa/verify-transcr) и даже
> адверсариальные агенты — это **ИИ, проверяющий ИИ**. Единственный неИИ-якорь до найма
> носителя — **авторитетные изданные курсы и цитируемые ELT/UK-источники**. Натуральность и
> охват ЛЮБОГО нового/правимого текста сверять **с ними**, НЕ с ухом владельца и НЕ с «здравым
> смыслом» агента. Метод доказан: source-cited multi-agent сверка 246 строк → 2 правки (`0ab5e1f`).

## Дополнительные цитируемые источники (для проверки фраз выживания)

Для коллокаций/идиоматики и зон повышенной ответственности (деньги/права/NHS/безопасность)
вердикт подкреплять цитатой из: **British Council** (learnenglish.britishcouncil.org),
**Cambridge/Oxford Dictionaries**, **NHS** (nhs.uk), **gov.uk**. Британские реалии
(`chemist`/`till`/`scales`/`cooker`) защищать как корректный BrE, не «исправлять» под AmE.

## Как этим пользоваться (процедура)

1. Каждый новый/правимый текст урока (grammar-примеры, диалоги, quiz-опции, everyday, simple_ru)
   и разговорника прогонять **source-cited multi-agent** сверкой: каждый вердикт — с цитатой из
   одного из 5 источников (или доп. цитируемых); **второй проход** отклоняет ложные флаги
   (BrE-слова не трогать). Это метод `0ab5e1f`, а НЕ generic-ревью «на вкус агента».
2. Запуск процедуры — скилл `/lesson-naturalness-check <Ln>` (`.claude/skills/`).
3. Это гейт перед коммитом урока (шаг 3 в CLAUDE.md «Процесс работы»), вне `npm run audit`
   (сеть/агенты). Живой носитель + YouGlish-прослушка владельцем — финальный гейт перед
   публичным запуском (B1–B2).
