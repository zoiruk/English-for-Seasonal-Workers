# План: «Не понял? Объясню ещё проще» + YouTube-резерв — 2026-06-19

Фича: в конце таба **Грамматика** — свёрнутая помощь для тех, кто не понял правило.
Внутри-урочное объяснение (основной путь) + ссылка на поиск YouTube как **последний резерв**.
Решения приняты через grill-сессию 2026-06-19 (владелец).

## Зачем
Грамм-блок уже объясняет 4 способами (intro ❌/✅ · cultural · note · examples). Но если ученик
всё равно не понял — нужна (1) ещё более простая опора в приложении и (2) аварийный выход на
видео-объяснение. Аудитория: A0, нулевой английский, низкая цифровая грамотность, нестабильный
интернет. Поэтому видео — русскоязычное, и только как резерв за тогглом.

## Принятые решения (и почему)
1. **Основной путь — внутри приложения**, не сразу YouTube. Сохраняет offline-first и контроль контента.
2. **Контент «ещё проще» — новое поле `grammar.simple_ru`:** «формула» (сворачивание темы в ОДИН
   шаблон, напр. «Кто + am/is/are + слово») + **1–2 предельно простых примера** с разбором по словам.
   Это не повтор правил, а другой угол. Пишется под урок, проходит аудит/ревью.
3. **YouTube — последний резерв**, НЕ выверенный ролик, а **ручной русский поисковый запрос**
   (`grammar.ytQuery`, напр. «present continuous простыми словами для начинающих»). Контролируем
   формулировку — единственный рычаг на неконтролируемом механизме.
4. **UX — одна свёрнутая лесенка эскалации** (НЕ две кнопки, НЕ раскрыто по умолчанию):
   кнопка «🤔 Не понял? Нажми» → раскрывает: формула + мини-примеры (с 🔊 TTS), а В НИЗУ той же
   панели — мелкая ссылка «Всё ещё непонятно? Видео на YouTube ▶ (нужен интернет)».
5. **Инструкция возврата** (обязательно): «Посмотри видео и вернись — нажми ‹ Назад и продолжай урок».
6. **Открытие ссылки:** `target="_blank"` (новая вкладка) — PWA остаётся открытым позади, возврат в один
   тап; прогресс и так в localStorage.
7. **Офлайн:** статичная подсказка «(нужен интернет)» рядом со ссылкой; БЕЗ детект-логики
   (`navigator.onLine` ненадёжен). Формула/примеры работают офлайн всегда.
8. **Объём:** движок + `simple_ru`/`ytQuery` **только для L1** (эталон) + адверсариальное ревью текста.
   L2–L15 добавляют поля позже, по одному. Уроки без полей валидны → кнопка не показывается.

## Отклонённые варианты
- Сырой YouTube-поиск как основной путь (вместо внутри-урочного) — неконтролируемая выдача.
- Выверенный конкретный ролик на урок — владелец предпочёл поисковый запрос (резерв).
- Авто-запрос из `title_ru` — даёт кривой запрос с английским внутри («This is a / These are»).
- Свернуть существующие note/cultural без нового контента — «те же слова, что он уже не понял».
- Автоскрытие ссылки офлайн через `navigator.onLine` — ненадёжно, лишний код.
- Ретрофит всех 7 сразу — отложено; сначала эталон L1.

## Модель данных (новые опц. поля у `grammar`)
```js
grammar: {
  // … существующие intro_ru/cultural_ru/note_ru/forms/examples …
  simple_ru: {                       // опционально
    formula: "Кто + am/is/are + слово",   // строка/HTML, можно <span class="g-transcr">
    examples: [                      // 1–2 шт., снежок-чисты (как grammar.examples)
      { en: "I am a worker.", transcr: "Ай эм э уёкэ", ru: "Я рабочий." }
    ]
  },
  ytQuery: "глагол to be простыми словами для начинающих"  // опционально, ручной RU
}
```

## Движок (`app/app.js`)
- В `grammarBody(les)` — после карточки examples — если есть `g.simple_ru`, дорисовать свёрнутый блок:
  кнопка-тоггл (по умолчанию closed) → панель: формула + мини-примеры (рендер как `ex-row` + `spkBtn`)
  + (если есть `g.ytQuery`) ссылка `https://www.youtube.com/results?search_query=` +
  `encodeURIComponent(g.ytQuery)`, `target="_blank" rel="noopener"`, с меткой «(нужен интернет)»
  + строка-инструкция возврата.
- Навесить toggle-обработчик и TTS на мини-примеры в `wireBody` (ветка `tab === "grammar"`).
- UI-строки — через `t()`/`UI.ru` (localization-ready): `help_toggle`, `help_yt`, `help_yt_offline_hint`,
  `help_return`.

## Аудит/снежок (`scripts/`)
- `snowball.js`: проверять `grammar.simple_ru.examples[].en` (как `grammar.examples`).
- `audit.js`: если `simple_ru` есть → `formula` непустая, `examples.length ≥ 1`, у каждого `en/transcr/ru`.
  `ytQuery` — опц. строка. Оба поля опциональны (отсутствие = валидно).

## UI/дизайн
- Telegram-light, тоггл-кнопка в стиле существующих (pill/card), 🔊 как везде.
- Ссылка YouTube — вторичная/мелкая (это резерв, не акцент). Touch target ≥44px.
